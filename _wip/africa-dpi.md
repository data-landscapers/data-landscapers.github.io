<section style="padding: 2.5rem 0; border-bottom: 1px solid var(--rule);">
  <h2>Overview</h2>
  <p>This study maps the digital public infrastructure landscape across all 54 African countries, covering five pillars: ICT infrastructure, digital identity, digital payments, data exchange, and registries.</p>
  <p>The study uses a novel methodology combining AI-powered search (Perplexity Computer with parallel 54-country agents) with structured metadata and human validation. It is the first continental-scale DPI assessment of this kind.</p>
  <p><a href="https://data-landscapers.github.io/africa-dpi/" class="btn" target="_blank">View live dashboard →</a></p>
</section>

<section style="padding: 2.5rem 0; border-bottom: 1px solid var(--rule);">
  <h2>Methodology</h2>
  <p>Data collection uses Perplexity Computer — an AI orchestration layer that distributes queries simultaneously to 54 independent agents, one per country. Each agent searches for a defined set of indicators using structured metadata. Claude Opus 4.6 distils results into structured CSV output.</p>
  <p>Full methodology documentation: <a href="https://data-landscapers.github.io/africa-dpi/manual/site/">data-landscapers.github.io/africa-dpi/manual</a></p>
</section>
